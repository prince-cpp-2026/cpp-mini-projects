# Calculator & Leap Year Checker

   Simple C++ programs for Day 2 TC course assignment.

   ## 📝 Description
   1. **calculator.cpp** - Performs +, -, *, / on two numbers. Handles divide-by-zero error and invalid operators.
   2. **leap_year.cpp** - Checks if a given year is a leap year or not.

   ## 🚀 How to Run

   **Requirements:** g++ compiler

   **Compile:**
   ```bash
   g++ calculator.cpp -o calculator
   g++ leap_year.cpp -o leap_year
